# 아마추어 골프 대회 - 신페리오 계산기

## 개요
`index.html` 하나로 동작하는 클라이언트 사이드 앱으로, 아마추어 골프 대회의 신페리오(Shin Peoria) 방식 순위를 계산합니다. Tailwind CSS CDN과 Font Awesome을 불러와 반응형 UI를 구성했고, 모든 데이터 상태와 계산 로직은 순수 JavaScript로 관리합니다.

## 주요 기능
- **코스/인원 설정**: 코스를 여러 개 정의하고 코스별 참가 인원 수를 입력하면 즉시 플레이어 목록이 생성됩니다. 모달 확인 후 삭제, 초기화가 가능하며 총 인원이 헤더에 집계됩니다.
- **파(Par)·히든홀 관리**: 각 코스별 1~18홀 파 값을 직접 수정할 수 있고, 버튼 한 번으로 Par4 8개 + Par3 2개 + Par5 2개 조합의 히든홀을 무작위 재추첨합니다.
- **플레이어 스코어 입력**: 코스별 그룹 헤더 아래에서 이름을 수정하고 18홀 스코어를 입력하면 총타수(Gross)가 즉시 반영됩니다. 필요 시 랜덤 채우기 버튼으로 더미 데이터를 생성할 수 있습니다.
- **신페리오 순위표**: 히든 12홀 점수로 핸디캡을 계산한 뒤 순위를 정렬하여 메달 아이콘과 함께 보여 줍니다.

## 계산식
리더보드 상단 안내와 동일하게, 히든 12홀 스코어를 1.5배한 값과 히든 Par 합을 활용해 핸디캡을 산출하고 Net 스코어를 계산합니다.

```202:205:index.html
                <p><strong>📊 계산 공식:</strong> (히든 12홀 × 1.5 - 히든 12홀 Par) × 0.8 = 핸디캡</p>
```

## 사용 방법
1. **코스 구성**: “코스 추가”로 그룹을 늘리고 이름/인원을 조정합니다. 삭제 시 모달에서 확인 후 진행됩니다.
2. **파 & 히든홀 세팅**: 탭으로 코스를 선택한 후 Par 값을 입력하고 “히든홀 재추첨”으로 조합을 갱신합니다.
3. **스코어 입력**: 플레이어 행에서 이름과 18홀 스코어를 채웁니다. 히든홀은 옅은 초록 배경으로 구분됩니다.
4. **결과 확인/인쇄**: 순위표에서 Net 스코어를 확인하고, 헤더의 인쇄 버튼으로 결과를 출력할 수 있습니다.

## 추가 메모
- DOMContentLoaded 시 기본으로 A코스(8명), B코스(4명)가 생성됩니다.
- 모든 데이터는 페이지 메모리에만 존재하므로 새로고침 시 초기 상태로 되돌아갑니다.
